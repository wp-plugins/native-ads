== Description ==

**Native ads.**

- Native cross-screen advertising for your site. Video plays when 50%+ viewable and can be contextually targeted by Channel or country. Demo: http://spoti.io
The ads come from : http://content.spoti.io/ and are geotargeted using http://target.spoti.io/

== Installation ==

- Install from WP Dashboard>>Plugins>>Add new plugin>>Search for 'Nativeads' plugin. Get your unique Client ID and configure the plug-in.
